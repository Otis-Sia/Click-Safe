Phone & Computer Basics: Protect Your Devices

Why Device Security Matters

Your phone and computer are gateways to all your personal information. Ensuring they're secure is crucial to protecting your identity, financial information, and privacy.

Understanding Device Threats

Malware & Chain Exploitation

Malware doesn't just affect one device; it uses social trust to spread. In the Chain Infection Model, one infected user can unknowingly infect their entire network of contacts.

Key Techniques:

  • Drive-by Downloads: Malware that installs simply by visiting a compromised site.
  • Botnets: Infected devices turning into "zombies" controlled by attackers.
  • XSS Worms: Self-replicating scripts spreading through social platforms.

Essential Security Practices

Keep Software Updated

Updates include critical security patches that fix vulnerabilities. Don't ignore update notifications.

  • Enable automatic updates in your device settings
  • Update your operating system regularly
  • Keep all applications up-to-date
  • Update browser and plugins

Use Antivirus & Anti-Malware

Protect your devices from malicious software:

  • Install reputable antivirus software (Windows Defender, Norton, McAfee)
  • Run regular scans for malware
  • Keep antivirus definitions updated
  • Be cautious when downloading files from unknown sources

Enable Device Locks

Prevent unauthorized physical access to your devices:

  • Set a strong PIN or password for device unlock
  • Use biometric security (fingerprint, face recognition)
  • Enable lock screen that activates after 5 minutes of inactivity
  • Lock your computer when stepping away

Safe Browsing Habits

  • Check for HTTPS (lock icon) before entering sensitive information
  • Avoid clicking links in unsolicited emails or messages
  • Download files only from trusted sources
  • Don't open email attachments from unknown senders
  • Use a password manager to auto-fill credentials (avoids phishing sites)

Network Security

  • Use a strong password for your home Wi-Fi router
  • Avoid public Wi-Fi for sensitive transactions
  • Use a VPN (Virtual Private Network) on public networks
  • Disable auto-connect to Wi-Fi networks
  • Keep your router firmware updated

What to Do If Compromised

  1. Disconnect from the internet immediately
  2. Change all important passwords from a different device
  3. Run a full antivirus scan
  4. Check for unauthorized accounts or changes
  5. Consider professional help if needed
  6. Monitor accounts for fraudulent activity

Quick Checklist

  • Enable automatic updates
  • Install antivirus software
  • Set up device lock with strong PIN/password
  • Use HTTPS and check for secure connections
  • Secure your Wi-Fi with a strong password
  • Consider using a VPN for public networks
