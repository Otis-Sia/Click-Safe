    3. Your New 3-Step System

    Step 1: Create a "Recipe"

    Pick a pattern only you know:

    [Color] + [Animal] + [Action] + [Time]

    Example: blue-elephant-swims-3pm

    Step 2: Make It Unique for Each Site

    Add the first 3 letters of the website:

    • Facebook: blue-elephant-swims-3pm-fac
    • Gmail: blue-elephant-swims-3pm-gma

    Step 3: Never Type It Again

    Use a password manager (more on this next).

    5. Two-Factor Authentication (2FA) Explained

    Think of it as two locks instead of one:

    • Your password (something you know)
    • Your phone/app (something you have)

    Setup takes 2 minutes:

    1. Go to security settings (Gmail, Facebook, etc.)
    2. Click "Enable 2FA"
    3. Scan the QR code with an app
    4. Save backup codes somewhere safe

    Best 2FA apps: Authy or Google Authenticator (both free)
