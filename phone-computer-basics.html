Essential security practices for your devices

Securing Your Devices

Your phone and computer contain sensitive personal information. Protecting these devices is crucial for maintaining your digital security.

Essential Security Settings

  • Keep software updated: Install operating system and app updates promptly
  • Use device encryption: Enable full-disk encryption on computers and phones
  • Set up screen locks: Use PIN, password, fingerprint, or face recognition
  • Enable auto-lock: Set devices to lock after a few minutes of inactivity
  • Install antivirus software: Use reputable security software on your computer

Safe Browsing Practices

  • Use HTTPS: Look for the padlock icon in your browser's address bar
  • Avoid public Wi-Fi for sensitive tasks: Don't access banking or enter passwords on public networks
  • Use a VPN: Encrypt your internet connection, especially on public Wi-Fi
  • Clear browsing data: Regularly delete cookies, cache, and browsing history
  • Be cautious with downloads: Only download from trusted sources

Mobile Security Tips

  • Download apps only from official app stores
  • Review app permissions before installing
  • Enable "Find My Device" features
  • Back up your data regularly
  • Disable Bluetooth when not in use
  • Use biometric authentication when available

Recognizing Threats

  • Phishing emails: Be suspicious of unexpected emails asking for personal information
  • Fake websites: Check URLs carefully before entering credentials
  • Pop-up warnings: Legitimate antivirus software doesn't use scary pop-ups
  • Suspicious links: Hover over links to see the actual destination
